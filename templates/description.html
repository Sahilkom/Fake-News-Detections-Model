Machine Learning Model Overview


The goal of this model is to classify news articles as either "Real" or "Fake" using a logistic regression classifier. The text data from news articles is processed and converted into numerical features using TF-IDF (Term Frequency-Inverse Document Frequency) vectorization. This allows the machine learning algorithm to understand and make predictions based on the text data.


Workflow of the Program


    • Importing Libraries:

    • The program starts by importing necessary libraries like NumPy, Pandas, NLTK, and scikit-learn. These libraries are used for data manipulation, natural language processing (NLP), and machine learning.

    • Loading and Preparing the Dataset:

    • The program mounts Google Drive to access the dataset (train.csv), which contains news articles labeled as "Real" or "Fake".
    • The dataset is loaded into a Pandas DataFrame (news_dataset), and its shape is checked to understand its size.
    • Missing values in the dataset are identified and replaced with empty strings to handle data cleanliness.

    • Text Preprocessing:

    • A new column (content) is created by combining the 'author' and 'title' columns of the dataset. This new column will be used as the input data for the model.
    • The stemming function is defined to preprocess the text:
      • It removes non-alphabet characters using regular expressions.
      • Converts the text to lowercase.
      • Splits the text into individual words.
      • Applies stemming using the Porter Stemmer to reduce words to their base or root form.
      • Removes stopwords (commonly occurring words like "and", "the", etc.) from the text.

    • Feature Extraction (TF-IDF Vectorization):

    • The TF-IDF vectorizer (TfidfVectorizer) from scikit-learn is used to convert the preprocessed text data (content) into numerical feature vectors.
    • TF-IDF assigns weights to words based on their importance in each document (news article) relative to the entire dataset. This helps in capturing the significance of words in distinguishing between different articles.

    • Splitting the Dataset:

    • The dataset is split into training and testing sets using train_test_split from scikit-learn. This allows evaluation of the model's performance on unseen data.

    • Training the Logistic Regression Model:

    • A logistic regression model (LogisticRegression) is instantiated and trained on the training data (X_train and Y_train).
    • Logistic regression is a classification algorithm that models the probability of a binary outcome (in this case, "Real" or "Fake" news).

    • Model Evaluation:

    • The accuracy of the trained model is evaluated on both the training and testing datasets using accuracy_score from scikit-learn.
    • The accuracy score helps assess how well the model is performing in terms of correctly predicting the news article labels.

    • Making Predictions:

    • After training, the model is used to make predictions (X_test_prediction) on the test data (X_test).
    • A sample news article (X_new) is also used to demonstrate how the model can predict its label (Real or Fake).

    • Saving the Model and Vectorizer:

    • Finally, the trained model (model) and TF-IDF vectorizer (vectorizer) are saved using pickle for future use or deployment.

Summary:


This program showcases a typical workflow for text classification using machine learning:


  • Data loading and preprocessing.
  • Text cleaning and feature extraction.
  • Model training and evaluation.
  • Prediction and model saving.

It demonstrates how to leverage NLP techniques and logistic regression for binary classification tasks, focusing on distinguishing between real and fake news articles based on their textual content.
